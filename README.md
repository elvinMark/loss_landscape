# Loss Landscape
## About
Visualizing loss landscape for some simple neural networks.

## Requirements
- PyTorch (>1.8)
- numpy
- scikit-learn
- matplotlib
- wandb

## Usage
```
python3 train.py \
--architecture MODEL_NAME \
--dataset DATASET_NAME 
```


```
python3 test.py \
--architecture MODEL_NAME \
--dataset DATASET_NAME 
```


```
python3 landscape.py \
--architecture MODEL_NAME \
--dataset DATASET_NAME 
```
